# 春笺 SpringPaper 首次使用说明

> AI 提高效率，判断仍属于老师。

这份说明只保留第一次使用最关键的步骤。正式阅卷前，建议先用 1 到 3 份试卷测试，确认截图、评分、点击和提交都正常。

## 1. 填写 API

打开 SpringPaper 后，进入“AI与标准”。

填写：

- API 供应商
- Base URL
- API Key
- 模型名称

如果使用硅基流动，请选择支持视觉输入的模型。

## 2. 导入评分标准

在“AI与标准”中填写或粘贴：

- 作文题目
- 评分标准
- 本次阅卷的特别说明

如果有题目截图、评分细则截图，也可以作为材料图片添加。

## 3. 校准作文区域

进入“校准”。

点击“校准作文区域”，拖选智学网页面中作文扫描件的主体区域。

建议把左右两页作文都框进去，避免模型漏看内容。

## 4. 校准分数按钮

继续在“校准”中完成：

- 校准分数网格
- 记录提交按钮

如果浏览器缩放、页面布局、显示器位置发生变化，需要重新校准。

## 5. 开始阅卷

回到“阅卷”。

先点击“截图并评分”，人工检查 AI 建议分是否符合你的判断。

确认无误后，再点击“采纳并提交”。

稳定后，可以使用“自动连评”。遇到低置信、疑问卷或明显异常时，及时停止并人工复核。

## 隐私提醒

- 不要把 API Key 发给别人。
- 不要把 `config.json`、`logs/`、试卷截图上传到公开网络。
- AI 输出只是建议，最终评分仍由老师负责。
