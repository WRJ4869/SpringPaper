# SpringPaper Roadmap

## v1.0.0

首个 Release-ready 版本。

- AI 作文建议评分。
- 智学网页面截图。
- 半自动提交。
- 自动连评。
- 人工复核。
- 区域校准。
- 本地轻量日志。
- CustomTkinter UI。
- SpringPaper 品牌命名与 About 弹窗。

## v1.0.x

小修小补，不新增大功能。

- 修复滚动、缩放、按钮遮挡等可用性问题。
- 优化日志可读性。
- 调整文案与 README。
- 补充使用手册和截图。

## v1.1.0

提升稳定性与可观察性。

- 增加更清晰的运行状态。
- 增加模型耗时、置信度、暂停原因统计。
- 优化疑问卷人工处理流程。

## v1.2.0

扩展使用场景。

- 支持更多题型或学科的评分模板。
- 支持多套评分标准管理。
- 支持更方便的模型切换和模型对比。

## Future

- Focus Mode：长时间阅卷时隐藏非必要信息。
- PyInstaller 正式打包。
- GitHub Release 自动化。
- 更完整的教师使用手册。
